
# 🧾 InvoiceGuard
### AI-Powered Invoice Classification System

A group coursework project using a Multi-Layer Perceptron (MLP) to classify
scanned business invoices into 4 categories:
**Utilities | Travel | Office Supplies | Entertainment**

## ⚙️ Setup

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Add your invoice images
Place images in subfolders under `data/raw/`:
```
data/raw/utilities/      → electricity bills, water bills, internet invoices
data/raw/travel/         → hotel receipts, flight tickets, fuel receipts
data/raw/office_supplies/→ stationery, furniture, equipment invoices
data/raw/entertainment/  → restaurant bills, event tickets, client meal receipts
```
Supported formats: `.jpg`, `.jpeg`, `.png`, `.bmp`, `.tiff`

### 3. Run notebooks in order
Open VSCode, install the Jupyter extension, then run each notebook:

| Step | Notebook | Who |
|------|----------|-----|
| 1 | `01_data_preprocessing.ipynb` | Member 1 |
| 2 | `02_feature_engineering.ipynb` | Member 2 |
| 3 | `03_model_training.ipynb` | Member 3 |
| 4 | `04_evaluation.ipynb` | Member 4 |

### 4. Launch the web UI
```bash
cd InvoiceGuard
streamlit run app/app.py
```
Then open http://localhost:8501 in your browser.

## 🧠 Model Architecture (MLP)

```
Input (raw pixels + handcrafted features)
        ↓
  Dense(512) → BatchNorm → ReLU → Dropout(0.3)
        ↓
  Dense(256) → BatchNorm → ReLU → Dropout(0.3)
        ↓
  Dense(128) → BatchNorm → ReLU → Dropout(0.2)
        ↓
  Dense(4)   → Softmax
        ↓
  [Utilities, Travel, Office Supplies, Entertainment]
```

## 🔬 Feature Engineering

Each image produces:
- **4,096** raw pixel values (64×64 grayscale)
- **1** edge density value (Canny edges)
- **16** pixel histogram values
- **HOG** features (shape/texture descriptors)
